# Quiz-Game 🎯
An interactive and responsive Quiz Game built using HTML, CSS, and JavaScript. This project features a modern UI with animated backgrounds, dynamic question handling, live score tracking, progress indicators, and a beautiful results dashboard.

 Features

* Responsive modern design
* Animated gradient background effects
* Dynamic quiz questions and options
* Real-time score tracking
* Progress bar indicator
* Multiple-choice support
* Circular score result animation
* Restart quiz functionality

 Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Google Fonts

Project Purpose

This project was created to improve frontend development skills and demonstrate interactive web application design using pure JavaScript without external frameworks.

 Future Improvements

* Timer for each question
* Difficulty levels
* Leaderboard system
* API-based questions
* Dark/Light mode support
